## Setup

1. Clone
	```
	git clone https://github.com/jgrazian/AdventOfCode2020.git
	cd AdventOfCode2020
	```

2. Create a config.json and add a SESSION_ID entry with your AoC input's cookie id.

3. Run
	```
	deno run -A --unstable aoc.ts day01
	```

4. Celebrate!
